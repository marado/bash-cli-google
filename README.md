# bash-cli-google
Command line Google search

## Install
<pre lang="bash">
<code>
brew install curl html-xml-utils
</code>
</pre>

## Usage
<pre lang="bash">
<code>
./search Ilari M채kel채
</code>
</pre>

If you wan't want color on the output, you can set the NOCOLOR variable:
<code>
NOCOLOR=1 ./search Ilari M채kel채
</code>
</pre>
